# Vegarely

Vegarely is an canvas editor ala Vasarely.

You can run it with Processing / Python Mode.

# Commands

*SPACE* : cycle effect

*UP* and *DOWN* : change effect range

*LEFT* and *RIGHT* : change effect strength

*d* : Cycle distance

*Right click* : Persist effect

*p* : "Print"in pdf

*c* : clear grid

*q* : Quit
